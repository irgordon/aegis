# AEGIS v0.4.1 Developer Preview

This is a draft GitHub Release for maintainer review.

It is a developer preview.

The artifacts are unsigned and not notarized.

They are local-only, pre-alpha, and developer-oriented.

They are not production-ready or enterprise-hardened.

They are archive-based. There is no installer.

There is no auto-update.

Validate SHA-256 checksums before use:

```bash
shasum -a 256 -c SHA256SUMS
```

Expected draft assets:

- `aegis-v0.4.1-macos-arm64.tar.gz`
- `aegis-v0.4.1-macos-x64.tar.gz`
- `SHA256SUMS`

Do not treat these artifacts as trusted production distribution.
