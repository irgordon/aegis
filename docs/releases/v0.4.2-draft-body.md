# AEGIS v0.4.2 Developer Preview Refresh

This release refreshes the first public AEGIS Developer Preview under the Release Truth invariant.

It remains a developer preview.

The artifacts are unsigned and not notarized.

They are prerelease, archive-based, and developer-oriented.

They are not production-ready or enterprise-hardened.

They are archive-based. There is no installer.

There is no auto-update.

The refresh includes:

- explicit latest-release and current-development identity
- a bundled safe `health.check` request fixture
- conventional gateway `--help` output
- neutral desktop no-error state
- first-run gateway guidance

Validate SHA-256 checksums before use:

```bash
shasum -a 256 -c SHA256SUMS
```

Expected assets:

- `aegis-v0.4.2-macos-arm64.tar.gz`
- `aegis-v0.4.2-macos-x64.tar.gz`
- `SHA256SUMS`

The historical `v0.4.1` tag and release remain unchanged.

Do not treat these artifacts as trusted production distribution.
